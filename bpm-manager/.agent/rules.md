# Reglas del Proyecto

1. **Gestión de Git**: 
   - NUNCA realices un `git push` de forma automática.
   - NO uses la frase "Ya hice el push a GitHub" o similares.
   - Limítate a realizar cambios locales y, si es necesario, prepara los commits, pero deja que el usuario decida cuándo subir los cambios al repositorio remoto.
